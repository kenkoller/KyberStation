# Launch Plan

KyberStation v1.0 launched on **2026-05-01** as a static GitHub Pages deployment at
<https://kenkoller.github.io/KyberStation/>.

This file exists as an acknowledgment that the launch has happened. The original
launch plan was developed in conversation rather than as a standalone document.

## What shipped at launch

- Visual blade style editor with 33 styles, 22 effects, 19 ignitions, 13 retractions
- AST-based ProffieOS 7.x codegen (compiles unmodified in Arduino IDE)
- Xenopixel V3 configuration mode
- 455+ presets (canonical + pop-culture)
- Modulation routing v1.1 with 11 modulators
- Saber Card snapshots (5 themes, 4 layouts, animated GIF export)
- Kyber Code sharing (base58 URL fragment)
- Sound font manager with SmoothSwing preview
- Offline-first PWA, no login, no backend
