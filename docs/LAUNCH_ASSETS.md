# Launch Assets

KyberStation launched on **2026-05-01**. This file documents the launch assets that
were used or planned.

## Reddit launch post

Posted to r/lightsabers on launch day. Response included feedback from community
member gschram92 about Fett263 config import, which drove the v0.17.0-v0.20.0
import sprint (14 PRs, 4 minor version bumps).

## Screenshots and media

Launch screenshots and GIFs were captured directly from the running application.
Animated GIF export is built into the My Crystal panel (idle hum loop + ignition
cycle variants).

## Post-launch outreach

YouTube outreach to saber content creators was planned but not yet executed.
Community feedback has been the primary growth channel.
